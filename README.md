# TVMaze – Práctica parcial (Cliente, 2025/26)

Aplicación de una sola página que consume el API público de TVMaze para mostrar series de TV, con búsqueda, detalle en modal y gestión de favoritos persistida en `localStorage`.

## Enunciado cumplido

- Campo de búsqueda al iniciar.
- Resultados con imagen y nombre de la serie.
- Detalle al hacer clic, mostrado en modal.
- Botón de favorito por serie para añadir/quitar de la lista
- Sección de favoritos siempre visible.
- Persistencia de favoritos en `localStorage`
- Toda la funcionalidad en una sola página usando modales.
- Documentación del API utilizada: https://www.tvmaze.com/api

## Desarrollo libre

- Sección Hero en la página principal mostrando una serie aleatoria

## Ejecución

```bash
npm install
npm start
```
